## Installation

### Clone the Repository

```bash
git clone https://github.com/Aathibarath/BlinkCart.git
```

### Navigate to the Project

```bash
cd BlinkCart
```

### Install Dependencies

```bash
npm install
```

### Start the Server

```bash
node server.js
```

or

```bash
npm start
```

### Open the Application

Open your browser and visit:

```
http://localhost:3000
```
